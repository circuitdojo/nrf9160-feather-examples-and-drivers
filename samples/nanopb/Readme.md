# Protobuf Sample

## Install `protoc`

### Windows

```
choco install protoc
```

### Mac

```
brew install protobuf
```

### Ubuntu

The version of `protoc` in `apt` is sorely of out date. Use the instructions [here](https://lindevs.com/install-protoc-on-ubuntu/) to install `protoc`.
